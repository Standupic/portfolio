    ТВ-приставка «WIFIRE»

    Приставка Wifire - это самое современное телевидение у Вас дома! Никаких сложных настроек и лишних проводов. На вашем экране в любое время доступны телеканалы, фильмы, передачи и интерактивные сервисы.

    • Работает везде, где есть Интернет
    • Поддерживает НD видео
    • Подключение по WI-Fi
    • Программа передач на экране

    Технические характеристики

    Smart TV

    Теперь, чтобы смотреть беспроводное цифровое телевидение, обладателям телевизоров с поддержкой Smart TV не нужно покупать приставку, достаточно установить бесплатное для скачивания приложение Wifire TV в Smart TV, пройти простую регистрацию и выбрать подходящий пакет телеканалов.

    На большинстве телеканалов Вы сможете по своему усмотрению останавливать трансляцию, записывать понравившиеся телепередачи и осуществлять при желании их перемотку назад.

    Приложения доступно на платформах

    Приложение Wifire TV может быть несовместимо с некоторыми моделями указанных телевизоров. Вы можете уточнить доступность приложения для Вашей модели телевизора в службе поддержки

    Cмартфоны и планшеты

    Смотрите Wifire TV на своем планшете или смартфоне, скачав приложение в App Store или Google Play.

    Простой и понятный интерфейс поможет быстро сориентироваться в возможностях приложения и легко пользоваться его основными функциями:

    • Настраивайте ,
      видимость каналов
    • Управляйте эфиром: пауза, перемотка и просмотр  любимых передач за последние 3 дня 
    • Добавляйте каналы
      в Избранное
    • Делитесь интересным с друзьями
      в социальных сетях

    Приложения доступно на платформах

    Сайт wifire.tv

    Доступен для браузеров Chrome и Safari

    Для просмотра видео через браузер необходимо установить плагин и расширение для браузера.

    • Плагин для Chrome

      Расширения для Chrome

    • Плагин для Safari

    Установка плагина необходима для защиты лицензированного контента от пиратства. Это полностью безопасно для вашего компьютера. Плагин необходим для корректной работы видеоплеера. Без него невозможно воспроизводить видеопотоки, зашифрованные по требованию правообладателей контента.

    Минимальные системные требования: OC: Windows 7 / MacOS 10.6; Процессор: Core i3 @ 2.00GHz (или аналог); Видеопамять: 512MB; ОЗУ: 4GB. Корректная работа на компьютерах, не отвечающих им, не гарантируется.
